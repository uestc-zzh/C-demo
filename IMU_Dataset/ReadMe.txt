1. 包含加速度计的三轴数据和陀螺仪的三轴数据；
2. 数据集中包含十一种动作，分别为行走、慢跑、静坐、挥手、深蹲、开合跳、早安式弯腰、箭步蹲、跳跃箭步蹲、跳跃深蹲和抬高跳（针对运动量较大的动作，由于采集者的体能有差异，部分文件中会有动作之间会有休息阶段（做动作->休息->做动作），在休息阶段，采集者保持静止状态）；
3. 每个数据文件的格式如下：
	每个数据文件包含N行8列，其中N为采集样本数，8列依次分别为同一时刻陀螺仪的x,y,z轴数据、加速度计的x,y,z轴数据、此采样点的使用2个16位表示的时间戳（目前时间戳不是真实值，文件中全为0）。
4. 加速度计和陀螺仪数据的采样率：25Hz；
5. 加速度计的量程: 正负8G, data resolution: 4096/1G（一个重力加速度用数值4096表示）;
6. 陀螺仪的量程: 正负2000°/s，data resolution: 16.4/(1°/s)（每秒一度的转速用数值16.4表示）。
